Tutorial 1
------------

To use spectrally-regularised-LVMs, first install it using pip:

.. code-block:: console

   (.venv) $ pip install spectrally-regularised-LVMs

>>> import spectrally_regularised_LVMs as srLVMs
>>> scLVMs.LinearModel()
   (.venv) $ pip install spectrally-regularised-LVMs
